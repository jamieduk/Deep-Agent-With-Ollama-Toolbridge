
Tool-bridge


git clone https://github.com/oct4pie/toolbridge.git
cd toolbridge

npm install

cp env.example .env

gedit .env

# after you edit your env file (make sure its set to use ollama and the toolbridge on port 4000 like in the python script, then...

npm start


or use

./start.sh


to auto start this use crontab

crontab -e

@reboot sleep 290 && /home/<USERNAME-HERE>/Documents/Scripts/AI/toolbridge/start.sh

you will need to update the path in start.sh so it works correctly via crontab / cronjob!
